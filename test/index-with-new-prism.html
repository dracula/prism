Prism.js

Install

  
    $ cd 'path/to/your_website_project'

Download with...

  
    $ git clone https://github.com/dracula/prism.git

or...

  
    svn export https://github.com/dracula/prism/trunk/css/dracula-prism.css 'css/'
    svn export https://github.com/dracula/prism/trunk/js/prism.js 'javascript/'

Link your project to these two files and to Google Webfonts' PT Mono. Do not forget to add .htaccess in order to allow the WOFF fonts. jQuery isn't required.

  
    <!-- Paste this in head -->
    <link href="https://fonts.googleapis.com/css?family=PT+Mono&subset=cyrillic,cyrillic-ext,latin-ext" rel="stylesheet"> 
    <link rel="stylesheet" href="css/dracula-prism.css">

    <!-- Paste this at the end of the file -->
    <script src="js/prism.js"></script>
    <script type="text/javascript">
      Prism.plugins.NormalizeWhitespace.setDefaults(
        {
          'remove-trailing': true,
          'remove-indent': true,
          'left-trim': true,
          'right-trim': true,
          // 'break-lines': 10,
          // 'indent': 2,
          // 'remove-initial-line-feed': false,
          // 'tabs-to-spaces': 4,
          // 'spaces-to-tabs': 4
        });
    </script>

Using the theme

Add a snippet to your website like this and replace language-xxx by one of the supported languages.

  
    <pre>
      <code class="language-xxx line-numbers">
        your snippet
      </code>
    </pre>

If you want to limit the pre height to 300, 400, ..., 800px, you can add <pre class="limit-xxx"> ... </pre>:

  
    <pre class="limit-300">
      <code class="language-xxx line-numbers">
        your snippet
      </code>
    </pre>

Note that the customised scrollbar works only in the webkit-based browsers. If you want your scrollbars to be the same in all browsers, you can use:

  • Baron
  • jQuery Scrollbar
  • Malihu
  • NiceScroll
  • Perfect Scrollbar
  • React Custom Scrollbars
  • SimpleBar
