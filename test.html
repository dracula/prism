              <h3 class="titulo">Prism.js</h3>

              <h4 class="titulo">Install</h4>

              <pre>
                <code class="language-bash">
                  $ cd 'path/to/your_website_project'
                </code>
              </pre>

              <p>Download with...</p>

              <pre>
                <code class="language-bash">
                  $ git clone https://github.com/dracula/prism.git
                </code>
              </pre>

              <p>or...</p>

              <pre>
                <code class="language-bash">
                  svn export https://github.com/dracula/prism/trunk/css/dracula-prism.css 'css/'
                  svn export https://github.com/dracula/prism/trunk/js/prism.js 'javascript/'
                </code>
              </pre>

              <p>Link your project to these two files and to Google Webfonts' PT Mono. Do not forget to add
                <code class="language-none">.htaccess</code> in order to allow the WOFF fonts. jQuery isn't
                required.</p>

              <pre>
                <code class="language-html">
                  <!-- Paste this in head -->
                  <link href="https://fonts.googleapis.com/css?family=PT+Mono&subset=cyrillic,cyrillic-ext,latin-ext" rel="stylesheet"> 
                  <link rel="stylesheet" href="css/dracula-prism.css">

                  <!-- Paste this at the end of the file -->
                  <script src="js/prism.js"></script>
                  <script type="text/javascript">
                    Prism.plugins.NormalizeWhitespace.setDefaults(
                      {
                        'remove-trailing': true,
                        'remove-indent': true,
                        'left-trim': true,
                        'right-trim': true,
                        // 'break-lines': 10,
                        // 'indent': 2,
                        // 'remove-initial-line-feed': false,
                        // 'tabs-to-spaces': 4,
                        // 'spaces-to-tabs': 4
                      });
                  </script>
                </code>
              </pre>

              <h4 class="titulo">Using the theme</h4>

              <p>Add a snippet to your website like this and replace
                <code class="language-none">language-xxx</code> by one of the
                <a href="http://prismjs.com/#languages-list" name="List of Supported Languages" target="blank">supported languages</a>.</p>

              <pre>
                <code class="language-html">
                  <pre>
                    <code class="language-xxx line-numbers">
                      your snippet
                    </code>
                  </pre>
                </code>
              </pre>

              <p>If you want to limit the <code class="language-html">pre</code> height to 300, 400, ...,
                800px, you can add <code class="language-html"><pre class="limit-xxx"> ... </pre></code>
                :</p>

              <pre>
                <code class="language-html">
                  <pre class="limit-300">
                    <code class="language-xxx line-numbers">
                      your snippet
                    </code>
                  </pre>
                </code>
              </pre>

              <p>Note that the customised scrollbar works only in the webkit-based browsers. If you want
                your scrollbars to be the same in all browsers, you can use:</p>
              
              <ul>
                <li><a href="https://github.com/Diokuz/baron" target="blank">Baron</a></li>
